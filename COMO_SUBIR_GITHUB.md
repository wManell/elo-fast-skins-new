## 📥 OPÇÃO 2: VIA TERMINAL (SE VOCÊ SABE USAR GIT)

### Passo 1: Abrir Terminal
Abra o terminal na pasta onde extraiu o código

### Passo 2: Executar Comandos
```bash
# Configurar Git (primeira vez)
git config --global user.email "seu@email.com"
git config --global user.name "Seu Nome"

# Inicializar repositório
git init

# Adicionar arquivos
git add .

# Fazer commit
git commit -m "Sistema completo Elo Fast Skins"

# Adicionar remote
git remote add origin https://github.com/Dev-Senior82/elo-fast-skins-new.git

# Enviar para GitHub
git branch -M main
git push -u origin main
```

Se pedir usuário e senha:
- **Username:** Dev-Senior82
- **Password:** Use um **Personal Access Token** (não a senha)
  - Crie em: https://github.com/settings/tokens
