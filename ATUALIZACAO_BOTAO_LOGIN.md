## 🚀 COMO ATUALIZAR SEU SITE NA VERCEL

### **OPÇÃO 1: Via GitHub (Automático)** ⭐

1. Baixe o novo código atualizado (ZIP atualizado)
2. Extraia
3. Copie TODOS os arquivos
4. Cole na pasta do projeto no seu PC (substitua os antigos)
5. No terminal, execute:

```bash
git add .
git commit -m "Adiciona botão Login Booster"
git push origin main
```

6. **A Vercel vai atualizar AUTOMATICAMENTE!** 🎉
7. Aguarde 1-2 minutos
8. Acesse seu site e veja o botão!

---

### **OPÇÃO 2: Via Vercel Dashboard (Manual)**

1. Acesse: https://vercel.com
2. Entre no seu projeto: `elo-fast-skins-new`
3. Vá em **"Deployments"**
4. Clique em **"Redeploy"** no último deploy
5. Aguarde o rebuild
